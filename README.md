# 🌐 六角部落格模板練習 

 個人首頁 & RWD

這是我在六角學院練習完成的部落格模板作業。  
作品包含首頁排版、文字調整、RWD 響應式設計。

## 🔗 Demo

👉 線上預覽網址：https://scarlett-terra.github.io/Blog-Card/

---

## 🚀 使用技術

- HTML5 / CSS3  
- RWD 響應式設計  
- Flex  排版  
- Figma ( 理解文字距離、字體大小、內容一切排版要求）

---

## 📌 功能列表

### 1️⃣ 首頁排版
- 虛擬工程師 Alyse 自我介紹  
- 文字排版、版面置中、內文欄位設定  
- Header & Footer & Content設計  

### 2️⃣ 動態互動效果
- 按鈕點擊時變換顏色  
- 選單 hover / active 效果  

### 3️⃣ 內容管理
- 文章內文位置置中  
- 版面與文字視覺調整  

### 4️⃣ RWD 響應式設計
- 手機、平板、桌面版可正常瀏覽  
- 版面元素自動調整適應不同裝置
